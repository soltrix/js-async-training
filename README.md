# Сценарий обучения: Промисы, async/await (JavaScript)

## Цель
Короткий практический сценарий для обучения начинающего веб‑разработчика основам работы с асинхронностью в JavaScript: промисам, цепочкам `.then()/.catch()`, `Promise.all` и `async/await`.
Материалы-ориентир: https://learn.javascript.ru/async

## Структура занятия (45–60 минут)
1. Введение (5 мин)
   - Что такое асинхронность, почему она нужна (I/O, таймеры, сетевые запросы).
   - Сравнение колбэков и промисов — проблемы «callback hell».

2. Промисы — теория и простые примеры (10–15 мин)
   - Конструктор `new Promise((resolve, reject) => ...)`.
   - Состояния промиса: pending → fulfilled / rejected.
   - Практика: превращаем callback в промис.

3. Цепочки `.then()` и обработка ошибок (5–10 мин)
   - Возврат значения и следующая обработка.
   - `catch()` — где ловить ошибки.

4. `Promise.all` и параллельная работа (5 мин)
   - Как ждать несколько промисов одновременно.

5. `async/await` — синтаксический сахар (10–15 мин)
   - `async function`, `await` и обработка ошибок через `try/catch`.
   - Примеры: последовательный и параллельный вызов с `Promise.all`.

6. Мини-задание (домашнее): написать функцию, которая загружает несколько ресурсов параллельно и возвращает результат только когда все загружены (использовать `Promise.all` и `async/await`).

## Примеры кода
См. файл `examples.js` в репозитории — краткие, понятные примеры:
- `loadScriptPromise(src)` — конвертация callback API в промис.
- `delayedValue(ms, value)` — промис на `setTimeout` + `Promise.all` пример.
- `async`/`await` демонстрация и обработка ошибок.
